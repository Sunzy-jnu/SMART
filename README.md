# SMART
SMART is a spatially Regularized Deconvolution Model with Smoothness Constraints that simultaneously estimates cell type proportions and cell-type-specific gene expression for each spot in spatial transcriptomics.
# Background
SMART is a spatial transcriptome deconvolution model based on spatial smoothing regularization.
